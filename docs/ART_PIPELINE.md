# AI 生图美术管线（Art Pipeline）

> 状态：方案草案，待功能开发接近完成后落地。
> 目标：**用 AI 生成全部 2D 美术素材**，单条管线覆盖卡面/图标/立绘/特效，控制成本与风格一致性。
> 音乐素材暂不考虑，本文档只覆盖图片。

---

## 0. 一句话定位

**智谱 CogView 负责草图与简单素材（免费/低成本批量），Seedream 4.5 或 GLM-Image 负责精修与一致性敏感素材。** 两个模型分工，成本可控、质量有保障。

---

## 1. 为什么是这套组合

| 角色 | 模型 | 职责 | 价格 |
|---|---|---|---|
| **草图/批量** | 智谱 **CogView-3-Flash** | 风格探索、图标/特效等批量素材、草稿迭代 | **免费** |
| **精修/一致性** | 字节 **Seedream 4.5**（首选）或 智谱 **GLM-Image**（¥0.1/张备选） | 羁绊卡面、英雄立绘、Boss 等对风格统一与角色一致性要求高的素材 | Seedream ~¥0.20/张；GLM-Image ¥0.1/张 |

**分工理由**：
- CogView 免费、量大，适合"先出 20 张挑 3 张"的高淘汰率场景（图标、特效、草稿）。
- Seedream 4.5 的**主体锁定技术 + 统一生成/编辑架构**解决了全项目最难的两点：32 张羁绊卡风格统一、英雄多角度一致、后期局部重绘改细节。
- 若追求极致省钱，精修环节也可全部用 **GLM-Image（¥0.1/张）**，总成本再降一半。

## 2. 素材清单与成本估算

| 素材类型 | 具体内容 | 数量 | 迭代系数 | 生成张数 |
|---|---|---|---|---|
| 英雄立绘 | 主角多角度（正/侧/背）+ 皮肤 | 1 | 10× | ~10 |
| 敌人 + Boss | 小怪 ~8 + 精英 + 3 Boss | ~18 | 5× | ~90 |
| 羁绊卡面 | 8 套系 × ~4 张 | 32 | 4× | ~128 |
| 技能图标 | 技能槽 + 词条 | ~30 | 4× | ~120 |
| Boss debuff 图标 | 17 个 debuff | 17 | 4× | ~68 |
| 特效 | 弹道/暴击/火冰雷/变身/追击 | ~15 | 5× | ~75 |
| UI 元素 | 按钮/卡牌框/血条/商店 | ~10 | 3× | ~30 |
| 场景背景 | 战斗房间（房间生存演进后） | 3 | 5× | ~15 |
| **合计** | | | | **~536 张** |

**成本估算**：

| 方案 | 精修模型 | 草图（CogView 免费） | 精修（~40% 张数） | 总成本 |
|---|---|---|---|---|
| 最省心 | Seedream 4.5 | ¥0 | 214 张 × ¥0.20 | **~¥43** |
| 最省钱 | GLM-Image | ¥0 | 214 张 × ¥0.10 | **~¥21** |

> 草图阶段约 60% 张数走 CogView（免费），精修阶段约 40% 走 Seedream/GLM-Image。

## 3. 工作流

```
第1步：定风格（CogView 免费试错）
  → 文生图出 10-20 张探索图，挑出 3 张"风格锚图"
  → 风格锚图 = 1 英雄 + 1 羁绊卡 + 1 敌人
        ↓ 把最满意的一张存为【风格参考图】
第2步：批量草图（CogView 免费）
  → 图标、特效、debuff 图标等"高淘汰率"素材全部用 CogView
  → 每个 prompt 附带【风格参考图】
        ↓
第3步：精修与一致性（Seedream 4.5 或 GLM-Image）
  → 羁绊卡面（32 张要风格统一）→ Seedream 组图生成
  → 英雄多角度 → Seedream 主体锁定
  → Boss 立绘 → Seedream 文生图 + 图像编辑
        ↓
第4步：改细节（Seedream 图像编辑 / 局部重绘）
  → 改表情/加光效/换配色/换背景，不重新生成
        ↓
第5步：后处理（免费工具）
  → 去背景（remove.bg / 绿幕 keying）
  → 裁切 & 命名 → 入库 Godot
```

**关键原则**：第 1 步的"风格锚图"是全项目风格统一的基石。定好后，后续所有生成都引用它。

### 4.0 通用风格前缀（每条 prompt 都带）

```
[风格] 移动端2D游戏美术，半写实二次元画风，东方修仙玄幻主题，
浓郁宝石色调，金色镶边点缀，干净剪影，柔和轮廓光，

居中构图，适合抠图的纯色背景
```

## 6. 避坑清单

| 坑 | 解法 |
|---|---|
| AI 出的"像素画"其实是糊的 mixels | 不让 AI 出像素画；出高清立绘后自行缩放 |
| 同一角色不同图长得不一样 | 用角色参考图 + Seedream 主体锁定 + seed 锁定 |
| sprite sheet 帧之间角色抖动 | AI 不擅长；改用"单帧立绘 + Aseprite/Spine 骨骼动画" |
| 直接出透明背景有伪影 | 用纯色/绿幕背景 → remove.bg / FLUX Kontext 去背景 |
| 图标风格漂移 | 全部图标一个 prompt 模板 + 一个风格锚图 |
| 中文字乱码 | **绝不 让 AI 在图里写中文**；文字用 Godot Label 叠加 |

## 7. 后处理与入库 Godot

### 目录结构

```
client/src/assets/
├── characters/hero/        # 英雄立绘
├── enemies/                # 敌人 + boss
├── bonds/                  # 32 张羁绊卡
├── skills/                 # 技能图标
├── debuffs/                # 17 个 boss debuff 图标
├── effects/                # 弹道/暴击/元素
├── ui/                     # 按钮/框/血条
└── bg/                     # 战斗场景
```

### 命名规范

```
hero_<pose>.png              # hero_idle.png, hero_attack.png
enemy_<name>.png             # enemy_frost_wraith.png
boss_<debuff_id>.png         # boss_regen_core.png
bond_<set_id>_<index>.png    # bond_zhutian_01.png
skill_<id>.png               # skill_multishot.png
debuff_<id>.png              # debuff_blind_eye.png
icon_<category>_<name>.png   # icon_effect_crit.png
```

### 工作流

```
AI 出图 (JPG/PNG)
   ↓
去背景: remove.bg / Photoshop / FLUX Kontext inpaint  →  透明 PNG
   ↓
裁切 & 命名（按上表）
   ↓
(可选) 压缩: TinyPNG
   ↓
放入 client/src/assets/{对应子目录}/
   ↓
Godot import: 设置 filter/mipmap，精灵用 Sprite2D
```

## 8. 模型对比参考（决策依据）

| 模型 | 厂商 | 文生图 | 图像编辑 | 角色一致性 | 中文 prompt | 价格 | 本管线角色 |
|---|---|---|---|---|---|---|---|
| **CogView-3-Flash** | 智谱 | 良好 | 弱 | 一般 | 原生 | **免费** | 草图/批量 |
| **GLM-Image** | 智谱+华为 | 良好（文字渲染 SOTA） | 一般 | 一般 | 原生 | ¥0.1/张 | 精修（省钱备选） |
| **Seedream 4.5** | 字节 | ⭐双榜第一 | ⭐强项 | ⭐主体锁定 | 原生中英 | ~¥0.2/张 | 精修（首选） |
| 通义万相 wan2.6 | 阿里 | 良好 | 支持 | 一般 | 原生 | ¥0.2/张 | 备选 |
| Nano Banana 2 | Google | 优秀 | 优秀 | 优秀 | 英文为主 | 中 | （未采用） |
| GPT Image 1/2 | OpenAI | 优秀 | 优秀 | 优秀 | 良好 | 高 | （未采用） |

> 决策：CogView（免费草图）+ Seedream 4.5（精修首选）/ GLM-Image（精修省钱备选）。
> 如预算极紧，精修全部用 GLM-Image（¥0.1/张），总成本约 ¥21 搞定全部美术。

## 9. 落地步骤

1. **现在**：开发期间用色块 + Label 占位（当前状态），暂不出图。
2. **功能接近完成时**：
   - 去[智谱开放平台](https://open.bigmodel.cn/)开通 CogView，跑通免费草图。
   - 去[火山方舟](https://www.volcengine.com/product/doubao)开通 Seedream 4.5（或继续用 GLM-Image）。
3. **先出 3 张风格锚图**（英雄 + 1 羁绊卡 + 1 敌人），确定全游戏画风。
4. **批量产素材**：CogView 出图标/特效/debuff 草图 → Seedream 精修羁绊卡/英雄/Boss。
5. 全程预估成本 **¥21–43**。
