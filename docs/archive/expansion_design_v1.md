# 一、宠魅（召唤流）

## 1.1 世界观与境界体系

《宠魅》是御兽流开山作，主角**楚暮**养魂宠战斗。核心机制是**魂约**——与魂宠签订契约，随境界提升解锁更多魂约位。

**魂宠师 9 大境界**（对齐遮天 9 境）：

| 境界 | index | 魂约位 | 含义 |
|------|-------|--------|------|
| 魂徒 | 0 | 1 | 入门，第一只魂宠 |
| 魂士 | 1 | 2 | 初阶御兽师 |
| 魂师 | 2 | 3 | 可契约 3 只 |
| 魂主 | 3 | 4 | 中坚力量 |
| 魂皇 | 4 | 5 | 一方霸主 |
| 魂宰 | 5 | 6 | 顶级御兽师 |
| 魂朽 | 6 | 7 | 传说级 |
| 千年最强 | 7 | 9 | 千年一遇 |
| 万年强者 | 8 | 11 | 巅峰，万兽之主 |

> 每境界 3 念解锁 1 个魂约位；魂约位 = 召唤单位强化位（游戏化：羁绊 effect 累加到对应召唤单位）。

## 1.2 召唤兽三阶段设计（精简版）

> 原著考证：楚暮的第一魂约是白魇魔（被迫签订，寄宿在灵魂里），第四魂约才是莫邪（变异月光狐）。
> 天苍青蛰龙无法签约（振翅飞走），是序章震撼存在，不做召唤兽。
> 白魇魔会随楚暮变强而"半魔化"——楚暮驾驭白魇魔之力时进入半魔形态（原著第106章"半魔楚暮"），获得强大力量但有负面作用。

**核心思路**：全程只有 **3 个召唤单位**（莫邪 + 白魇魔 + 第三魂宠三选一），走精华路线。
召唤单位是**跟随英雄的虚拟单位**——有独立精灵图、跟随移动、独立攻击，但**不可被选中/攻击**。攻击强度随羁绊变强。

### 召唤单位系统（3 个虚拟跟随单位）

| 召唤单位 | 存在阶段 | 获得方式 | 强化方式 | 特殊机制 |
|----------|----------|----------|----------|----------|
| **莫邪** | 全程（境0-8） | 选宠魅体系自动获得 | 羁绊强化（进化换形态+精灵图），攻击力随羁绊升级 | 核心输出单位 |
| **白魇魔** | 前期~中期（境0-5） | 魂徒境界自动获得 | 羁绊强化（魔焰增强） | **被吞噬后消失**（魂朽境触发，力量融入主角=半魔化变身） |
| **第三魂宠**（三选一） | 后期（境6-8） | 魂朽境选一只 | 对应羁绊强化 | 小蛰龙（AOE）/ 善恶花王（控制）/ 缚风灵（速攻） |

**虚拟单位的实现**：
- 每个召唤单位是一个 `Node2D`，挂载在英雄节点下（跟随移动）
- 有独立精灵图（`Sprite2D`）+ 简单攻击动画
- 有独立的攻击间隔/伤害（来自羁绊 effect 累加）
- **不可被选中**：不进入敌人的目标选择，不受伤害（虚拟存在）
- 攻击时向最近敌人发射投射物（复用 projectile 系统）
- 羁绊强化 = 累加该单位的 atk/attack_speed/技能效果

### 莫邪的进化（随境界突破自动换形态 + 精灵图）

| 境界 | 莫邪形态 | 精灵图 | 攻击特性 |
|------|----------|--------|----------|
| 魂徒(境0) | 变异月光狐 | moonfox.png | 基础物理攻击 |
| 魂士(境1) | 六尾妖狐 | sixtail_evil.png | +邪焰属性伤害 |
| 魂师(境2) | 九尾炎狐 | ninetail_fire.png | +AOE（炎焰全开） |
| 魂宰(境5) | 七罪狐·堕落炎君 | sevensin.png | +真伤（罪炎） |
| 千年(境7) | 幽冥狐尊·紫帝 | zidi_purple.png | +最终伤害（紫焰冥炎） |

> 进化 = 境界突破时自动换形态（改精灵图 + 强化攻击属性），不需要单独的进化操作。

### 白魇魔的生命周期（特殊：被吞噬后消失）

| 阶段 | 白魇魔状态 | 主角状态 | 召唤单位数 | 玩法重心 |
|------|-----------|----------|-----------|----------|
| 魂徒~魂宰(境0-5) | **在场**（跟随单位） | 普通 | 莫邪 + 白魇魔 = **2 只** | 双召唤单位协同，白魇魔高伤但有反噬 |
| 魂朽(境6) | **被吞噬，消失** | 获得半魔化（**不稳定**：周期性变身，有冷却+变身后虚弱） | 莫邪 = **1 只** | 白魇魔消失的空档期，靠半魔化爆发过渡 |
| 千年(境7) | 不在场 | **莫邪进化紫帝** + **第三魂宠登场**（三选一）；半魔化仍周期性 | 莫邪(紫帝) + 第三魂宠 = **2 只** | 白魇魔的空缺被第三魂宠填补，玩法重新丰富 |
| 万年(境8) | 不在场 | **半魔化完全体**（常驻无冷却）+ 莫邪紫帝巅峰 + 第三魂宠终极 | 莫邪(紫帝) + 第三魂宠 = **2 只** + 主角常驻魔人 | 三合一巅峰：莫邪+第三魂宠+常驻半魔化 |

> **白魇魔消失后的节奏**（核心设计）：
> - 境6 吞噬 → 暂时只剩莫邪（阵痛期），但获得半魔化爆发
> - 境7 第三魂宠登场 → 填补白魇魔的空缺，重新回到 2 只召唤单位
> - 境8 半魔化常驻 → 主角自身也变成战力源（魔人形态），等于"3 个输出源"
>
> 这样白魇魔消失不会让后期变空——第三魂宠接力，且主角从"指挥者"变成"半参与者"（魔人常驻）。

### 第三魂宠（三选一，千年境登场）

到达千年境（境7）后——白魇魔已被吞噬，空出一个召唤位——玩家从三只后期魂宠中**选一只**填补：

| 第三魂宠 | 定位 | 进化终点 | build 走向 |
|----------|------|----------|------------|
| **小蛰龙→幽灵龙→天苍青蛰龙** | AOE 爆发（龙息） | 序章传说被驯服 | 群清流 |
| **善恶花王→善恶女王→善恶始祖** | 控制（花葬/眩晕） | 与雨娑融合 | 控制流 |
| **缚风灵→圣兽→七图圣兽** | 速攻（风系穿透） | 七图圣兽完全体 | 速攻流 |

> 选了一只 → 对应虚拟单位出现 + 对应羁绊进池；另两只永不出现。

### 半魔化的三个阶段（原著：无法控制 → 驾驭 → 常驻）

| 阶段 | 境界 | 半魔化状态 | 机制 |
|------|------|-----------|------|
| **觉醒** | 魂朽(境6) | 不稳定，无法完全控制 | 周期性变身（duration 8s / cooldown 60s），变身后虚弱 10s |
| **驾驭** | 千年(境7) | 可自由切换 | 冷却减半（cooldown 30s），无虚弱 |
| **常驻** | 万年(境8) | 完全融合，永久魔人 | 无冷却，常驻魔人形态，莫邪+第三魂宠攻击力翻倍 |

> 复用黑神话大圣变身框架（transform_mult/transform_duration/transform_cooldown）。
> 原著考证："楚暮无法控制半魔完全因为自己现在太过弱小，一旦自己实力足够，绝对可以随时随刻毫无顾忌的化身半魔"。

### 关于莫邪进化 / 白魇魔 / 第三魂宠（已合并进境界）

> 旧设计有"莫邪进化链 12 个 + 白魇魔链 8 个 + 第三魂宠 12 个"平行于境界羁绊——**大量重叠**。
> 新设计：这些魂宠的形态变化**直接作为境界子羁绊**（如"莫邪进化·六尾妖狐"在魂士境，"半魔楚暮·首次觉醒"在魂士境），不再平行。
> 这意味着：
> - 莫邪的进化 = 随境界突破自动发生（不需要单独的进化条件羁绊）
> - 白魇魔的觉醒/融合 = 随境界推进（魂徒签订→魂士半魔→魂皇稳定→魂朽融合）
> - 第三魂宠（小蛰龙/缚风灵/善恶花王）= 后期（魂朽境后）三选一获得，前期不出现

## 1.4 起点技能设计（1 个，随境界自动升级）

> 新设计下宠魅只有 **1 个起点技能**：召唤·莫邪。选了宠魅体系就自动获得。
> 莫邪的进化、白魇魔的觉醒、半魔化变身等，全部通过**境界突破的 reward（skill_upgrade）自动获得**，不再是独立抽取的技能。

```yaml
skills:
  # ══ 宠魅起点技能（选了宠魅体系自动获得，随境界自动升级）══
  - id: cm_summon_moxie             # 召唤·莫邪
    name: 召唤·莫邪
    rarity: SR                       # 起点
    tags: [summon, physical]
    atk_ratio: 0.8                   # 基础倍率
    note: 宠魅体系入口——选了才解锁宠魅羁绊池。随境界突破自动升级：
          魂士→莫邪进化六尾（+邪焰）
          魂宰→莫邪七罪狐（+真伤）
          千年→莫邪紫帝（+final_dmg）
          魂朽→半魔化变身（周期性爆发）
```

### 起点技能随境界自动升级（reward 里的 skill_upgrade）

```yaml

  魂士(境1):
    moxie: { sprite: sixtail_evil.png, effect: { atk_pct_delta: 0.15, elemental_dmg_mult: 0.10 } }   # 莫邪进化六尾妖狐
  魂师(境2):
    moxie: { sprite: ninetail_fire.png, effect: { atk_pct_delta: 0.10, attack_speed_delta: 0.10 } }  # 莫邪进化九尾炎狐
